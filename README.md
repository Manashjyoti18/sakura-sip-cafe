# 🌸 Sakura Sip Café

An anime-inspired Japanese café website featuring kawaii dining experiences with matcha lattes, soufflé pancakes, boba tea, and themed events.

## ✨ Features

- Beautiful anime-themed design with sakura petal animations
- Interactive menu with Japanese-style drinks, food, and desserts
- Online reservation system
- Light/Dark theme toggle
- Particle effects and smooth animations
- Fully responsive layout

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom animations, glassmorphism, gradients
- **JavaScript** — Interactive components, particle effects

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Manashjyoti18/sakura-sip-cafe.git
   ```
2. Open `index.html` in your browser — no build step required!
